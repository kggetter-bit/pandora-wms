# WMS Command Center

Prototype WMS web application built with React, Vite, Recharts, Three.js, and Lucide icons.

## Features

- Warehouse dashboard and KPI reports
- Inbound receiving with LPN, SN, and IMEI flows
- Putaway, move, prework/sticker, replenishment, allocation, picking, packing, and shipping flows
- Inventory transaction, overview, aging, stock cover, and total recall views
- Return, CS case, transfer, reservation, and check stock workflows
- Export-ready report views for Excel/PDF from the app UI

## Requirements

- Node.js 18+

## Install

```powershell
npm install
```

## Run

```powershell
npm run dev
```
